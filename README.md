+ Các class đặt tên có ý nghĩa

  - TextView: tv...
  - Button: btn...
  - Layout: ln...
  - ImageView: imv...
  - Checkbox: ckh...
  - CardView: card...
  - RecycleView: rcv...
  Tên cái resouce khác: 
  - icon : ic...
  - image: img...
  - layout: activity..., fragment..., layout...
  - layout cho một item của recycleView: item...
